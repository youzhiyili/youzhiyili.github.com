准备工具

下载Google App Engine SDK for Python,解压重命名为goa 下载goagent,解压到goa重命名为goagent

配置本地

cd到goa appid = appid_1|appid_2|appid_3|......|appid_10

上传服务器

cd到goa 上传appid_1 application: appid_1 python appcfg.py update goagent/server/python/ 上传appid_2 application: appid_2 python appcfg.py update goagent/server/python/ 上传appid_N application: appid_N python appcfg.py update goagent/server/python/

运行Goagent
